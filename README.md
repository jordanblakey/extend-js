# ffjs

Lightweight, modular utility functions for arrays, browser, dates, strings, math, and more.

## Installation

```sh
npm install ffjs
# OR
yarn add ffjs
```

## Usage

Everything:

``` js
import ff from 'ffjs'
// OR
const ff = require('ffjs').default
```

Single Module:

``` js
import { array } from 'ffjs'
const array = require('ffjs').array
```
